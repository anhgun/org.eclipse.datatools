How to run the recorded test cases
1. Install TPTP 4.4. By the time of writing this document, I'm using TPTP4.4 20070327 build.
2. Modify org.eclipse.datatools.sqltools.tests.verification.hooks\*.properties to fit your local environment.
2. Right click on test-resources\SQLToolsGUISuite.testsuite and choose "Run as -> Test"
