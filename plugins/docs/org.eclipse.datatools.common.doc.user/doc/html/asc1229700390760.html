Connecting to Apache Derby

Connect to an Apache Derby database from the Data Source Explorer.

A connection profile enables the runtime connection to endpoints for database development.

  1. In the Database Development perspective, expand Databases in the Data Source Explorer.
  2. Right-click the Apache Derby connection profile that you created and select Connect.

    After the initial contents for the database instance are created, which may take a few seconds, you can expand the connection profile and view the contents of the Apache Derby database.
