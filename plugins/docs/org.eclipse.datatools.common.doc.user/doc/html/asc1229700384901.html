Data Tools Platform Tutorial Overview

Use the tutorial to get started using the Data Tools Platform with Apache Derby.

The Data Tools Platform tutorial uses relational databases as its primary set of examples, and it uses Apache Derby as its sample database. When you complete the tutorial, you should understand the basic steps required to connect to and work with Apache Derby using the Data Tools Platform.

  1. Apache Derby Tutorial Prerequisites
    To use the tutorial, you must first install Eclipse, Data Tools Platform, and Apache Derby.
  2. Creating a Driver Definition for Apache Derby
    Define a driver to connect to Apache Derby.
  3. Creating an Apache Derby Connection Profile
    Create a connection profile to connect to Apache Derby.
  4. Connecting to Apache Derby
    Connect to an Apache Derby database from the Data Source Explorer.
  5. Running a Stored Procedure
    Run a stored procedure in the database using the Apache Derby connection.
  6. Creating and Executing a SQL Query
    Create a SQL file and run it against a given database instance.
