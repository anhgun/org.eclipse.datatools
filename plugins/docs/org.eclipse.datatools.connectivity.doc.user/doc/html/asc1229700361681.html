Connection Profile Default Schema Filter Properties

Indicate which schemas display in Data Source Explorer.

Property Description
Expression Select the filtering method.
Text box Enter one or more characters used in the filter expression. This field is case sensitive. You do not need to use wildcards or meta-characters.

For example, if you selected Starts With the Characters and entered a capital S in this text box, Data Source Explorer displays only the schemas with names that begin with a capital S.

Disable Filter Deselect the checkbox to enter a filter expression to determine which schemas display in Data Source Explorer.
