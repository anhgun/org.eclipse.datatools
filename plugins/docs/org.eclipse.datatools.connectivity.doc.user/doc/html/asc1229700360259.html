Connection Profile Common Properties

Edit the properties common to all connection profiles.

Property Description
Name Enter a unique and descriptive name for the connection profile.
Description (Optional) Enter a profile description that helps you distinguish it from others.
Connect Every Time the Workbench Is Started (Optional) Connect automatically to the data source when the workbench displays during startup.
