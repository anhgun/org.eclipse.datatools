Setting Connectivity Preferences

Set preferences for how nodes get displayed for Open Data Access, connection profiles and driver definitions.

  1. Select Window > Preferences from the main menu.
  2. Expand Sybase, Inc > Data Management > Connectivity and select Open Data Access, Database Connection Profile or Driver Definitions.
  • Database Connection Profile Preferences
    Determines how connection profile information displays: grouped by schema or listed alphabetically with database object owner in parentheses next to the object name.
  • Driver Definitions
    Add, edit, or delete driver definitions for a specific database server. Connection profiles use driver definitions to establish database connections.
