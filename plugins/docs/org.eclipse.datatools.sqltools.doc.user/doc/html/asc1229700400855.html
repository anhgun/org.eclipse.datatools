Database Development General Preferences

The general user preferences concern executing and debugging procedural objects.

Table 1. General preferences
Property Description
Show Affordance in Hover on How to Make it Sticky Displays the Press F2 for focus message in hover help (ToolTips). If you make a hover message sticky, the message text opens in a scrollable window from which you can select and copy content.
SQL Error Execution Action Select the action to be taken on an error during SQL execution.
  • Always — always continues execution when an error occurs.
  • Never — always stops execution when an error occurs.
  • Prompt — interrupts execution on each error, displaying a dialog to resume execution (default).
