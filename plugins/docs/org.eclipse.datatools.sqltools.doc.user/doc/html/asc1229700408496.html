Syntax Coloring Preferences

Indicate your preferences for syntax coloring and text style for SQL source code in SQL Editor or SQL Scrapbook.

Table 1. Syntax Coloring preferences
Field Description
Syntax Items Select a syntax item.
Color and Text Style Click the Color box to select a text color or select a text formatting style, or both, to determine how the selected syntax item is rendered in the editor. You can select multiple text formatting styles.
Preview Displays an example of the syntax coloring choices.
