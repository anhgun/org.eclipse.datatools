Saving a SQL File

Save one or more SQL files to the current project, or save a SQL file to a different project or to a different file name.

Save one or more SQL files simultaneously.
Option Description
Save to current project Select File > Save from the main menu bar.
Save to different project Select File > Save As from the main menu bar, and specify project information in the Save As dialog.
Save to different file name Select File > Save As from the main menu bar, and specify a file name in the Save As dialog.
Save all SQL files to the current project Select File > Save All from the main menu bar.
