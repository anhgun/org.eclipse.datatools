History Options Preferences

Set history option preferences for the display of history in the SQL Results view.

Table 1. SQL Results view history options
Property Description
Columns to Display Select the columns that comprise the history displayed in the SQL Results view.
Automatically Persist Result History Save the results history when WorkSpace shuts down; load the results history in the SQL Results view when WorkSpace starts up.
Automatically Clean Up Result History Clears result history when WorkSpace shuts down.
