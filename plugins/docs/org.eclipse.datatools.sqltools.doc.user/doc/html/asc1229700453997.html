SQL File Editor

Use the SQL File Editor to create and edit SQL statements.

The SQL File Editor enables standard text-based editing of SQL statements with the added functionality of Content Assist, syntax color, and multiple statement support. You can associate the SQL file with a specific connection profile, and use the SQL File Editor to graphically build SQL statements using Visual SQL, SQL Query Builder. as well as create procedural objects.The SQL File Editor also enables you to insert stored procedures and triggers, and to access to ASE real-time messaging and XML services.
